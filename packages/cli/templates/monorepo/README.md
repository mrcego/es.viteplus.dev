# Inicio de Monorepo Vite+

Un punto de partida para crear un monorepo con Vite+.

## Desarrollo

- Comprobar que todo esté listo:

```bash
vp run ready
```

- Ejecutar las pruebas:

```bash
vp run test -r
```

- Construir el monorepo:

```bash
vp run build -r
```

- Ejecutar el servidor de desarrollo:

```bash
vp run dev
```
